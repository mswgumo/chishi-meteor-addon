Ŝerca Meteor-kromaĵo - nur por amuzo

# Enkonduko
Tio estas Chishi-kromaĵo bazita sur MeteorClient

Necesas instali MeteorClient por uzi ĝin

# Instalado
1. Elŝutu Minecraft 1.21.11
2. Instalu Fabric
3. Elŝutu la koncernan version de MeteorClient kaj metu la dosieron en la dosierujon "mods"
4. Elŝutu la plej novan JAR-dosieron el Releases kaj metu ĝin en la dosierujon "mods"

# Funkcioj
| Modulo | Priskribo |
|--------|-----------|
| **NoJump** | Ne eblas salti |
| **NoSprint** | Ne eblas sprinti |
| **Boom** | Unu-klaka kraŝigo de la kliento |
| **ShortRange** | La ataka distanco mallongiĝas |
| **SelfDamage** | Post ataki enton, tuj rulas la komandon "kill"; necesas ke la servilo subtenu ĝin. En 3C3U uzas "suicide --confirm" |
| **InventoryShuffle** | Aŭtomate miksas la inventaron |
| **AntiAura** | Kontraŭ-aŭro: post atako aŭtomate forĵetas la armilon |
| **AntiTotem** | Kontraŭ-totemo: aŭtomate forĵetas la totemon el la malĉefa mano |
| **NoMine** | Ne eblas minadi blokojn; kiam aktivigita, blokoj ne povas esti minataj |
| **TalkNonsense** | Faras la ludanton paroli sensencaĵojn |
| FPSKiller | FPS Killer igas vian ludon tre malrapida |
| Genshin | Genshin ekfunkciis!!! |

# Konstruado
- Rulu `git clone https://github.com/mswgumo/chishi-meteor-addon.git`
- Rulu `cd chishi-meteor-addon`
- Rulu `./gradlew build`

- Uzu hack-klientojn kiel Meteor nur en senregulaj serviloj kiel 2B2T aŭ 3C3U; ne uzu ilin en normalaj serviloj
